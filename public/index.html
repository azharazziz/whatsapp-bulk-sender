Upload Kontak
Format yang didukung:
  • CSV/TXT: Nama,Nomor Telepon (satu per baris)
  • vCard (.vcf): File kontak dari HP/aplikasi kontak
Contoh CSV: "John Doe,081234567890"
File vCard dapat diekspor dari aplikasi kontak HP Anda
Cara ekspor vCard: Buka Kontak → Pilih kontak → Share/Bagikan → Pilih vCard/VCF
